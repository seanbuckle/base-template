# Contributing to this Project

Thanks for being here! To keep things smooth, please follow these guidelines.

## 🛠️ Local Setup

1. Clone the repo.
2. Install dependencies (standard for the language you choose).
3. Copy `.env.example` to `.env` and fill in secrets.

## 🌿 Branching Strategy

We use a prefix-based branching system:

- `feat/` for new features.
- `fix/` for bug fixes.
- `docs/` for documentation changes.
- `chore/` for maintenance.

## 🧪 Testing

Please ensure all tests pass before opening a Pull Request.
